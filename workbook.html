Background

This document presents an analysis of data from the European Social Survey (ESS), with a focus on self-reported life satisfaction across European countries.

The aim is twofold:

  1. To show how to move from a raw survey file to a clean and well-labelled analysis dataset using the tidyverse.
  2. To explore patterns in life satisfaction by key sociodemographic characteristics (such as age and gender) and across countries, using descriptive statistics, visualisations, and simple regression models.

Throughout, the emphasis is on a reproducible workflow that makes each step transparent and easy to adapt.

Before you start, please download the ESS data file in .csv format and place it in a folder that you can access from R. Make sure:

  • Your R working directory points to the folder containing the data file.
  • The path used in the code below matches where you actually saved the data.
